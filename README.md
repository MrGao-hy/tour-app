### 使用教程
##### 下载pnpm
```bash
$ npm i -g pnpm
```
##### 下载依赖包
```bash
$ pnpm install
```
##### 运行项目到h5
```bash
$ npm run dev:h5
```
##### 运行项目到微信小程序
```bash
$ npm run dev:mp-weixin
```
##### 运行项目到APP
```angular2html
推荐在HbuilderX编辑器运行，连接手机端或者模拟器
```


### 使用说明

1.  node版本推荐 14.19.*以上
2.  pnpm版本推荐 7**
